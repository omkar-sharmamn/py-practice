# py-practice

Simple and basic problems coded in python

1. factorial.py : Gives factorial of a number using recursion.
2. fib.py : Gives fibonacci of a number.
3. gen_prime.py : Generates prime numbers of given limit
4. is_prime.py : Checks whether a number is a prime number or not .
5. issum.py : Sums the digits in given number.
6. remove_dups.py : Remove duplicate element in an array
7. returning_cube.py : Returns cube of a number if the number is a multiple of 3 ;)
8. palindrome.py : Checks whether a string is palindrome or not Return True if yes else False 
9. sliding.py : Read window size and list of numbers from a file. Take only the numbers equal to window size and display smallest number in them. And remove the first element and add next element from the list and move on. Refer to slide.txt for window size and list of numbers.
10. string_reversal.py : Reverse a string and print it.
